# Modern E-Commerce Dashboard

This is a state-of-the-art e-commerce dashboard built with **Next.js** and **Tailwind CSS**. It provides store owners with real-time analytics, comprehensive order tracking, and an intuitive user interface.

## Key Features

- **Real-Time Analytics**: Built-in charts showing daily sales, customer growth, and conversion rates.
- **Order Management**: Detailed views for orders, invoices, and shipment tracking.
- **Vibrant Dark Mode**: Beautiful dark mode styled with Tailwind's native utilities.
- **Responsive Layout**: Works flawlessly on desktops, tablets, and mobile devices.

## Tech Stack Details

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Charts**: Recharts for responsive SVG visualization
- **State Management**: Zustand
- **Database**: PostgreSQL with Prisma ORM
