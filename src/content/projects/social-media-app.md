# Social Media Platform

A fully functional social network where users can share their thoughts, upload images, and interact with others in real-time.

## Key Features

- **Real-Time Chat**: Send messages instantly with Socket.io.
- **News Feed**: Infinite scrolling feed with the latest updates from your network.
- **User Profiles**: Customizable profiles with avatars and bio.
- **Secure Authentication**: JWT-based auth with password hashing.

## Architecture

Built using the MERN stack with a focus on scalable real-time capabilities.
