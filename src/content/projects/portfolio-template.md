# Developer Portfolio Template

Showcase your projects and skills with this beautiful, fast-loading Astro template.

## Features

- **Blazing Fast**: Static generation with Astro ensures 100/100 Lighthouse scores.
- **MDX Support**: Write your blog posts and project details in Markdown or MDX.
- **SEO Optimized**: Built-in meta tags, OpenGraph, and sitemap generation.
- **Accessible**: Semantic HTML and ARIA labels.

## Getting Started

Just clone the repository, update the configuration file, and start writing!
